# Oh! I Forgot 📱
### Never forget where you kept things at home

---

## HOW TO INSTALL ON ANDROID

This is a **Progressive Web App (PWA)** — it installs like a native app, works offline, and stores all photos locally on your phone.

Install on Android

1. Open Chrome on your Android phone
2. Navigate to your app URL
3. Chrome will show a banner: **"Add Oh! I Forgot to Home screen"** — tap it
4. OR tap the 3-dot menu → **"Add to Home screen"**
5. The app will appear as an icon on your home screen just like a native app!

---

## FEATURES

✅ Add rooms (Bedroom, Kitchen, Garage, etc.) with custom icons
✅ Add sub-locations inside each room (Drawer, Shelf, Wardrobe...)
✅ Take/upload photos of items, automatically compressed to save space
✅ Search by item name, room, tags, or description
✅ Set reminders with date & time — get push notifications
✅ Mind Map view showing all rooms and items visually
✅ Share item location via any app (WhatsApp, SMS, Email...)
✅ Works 100% offline — all data stored locally on your phone
✅ Photos compressed to ~70% quality at max 800px — saves storage

---

## DATA & PRIVACY

- **All data is stored locally** using the browser's localStorage
- **Photos are stored locally** — compressed before saving
- **Nothing is uploaded to any server**
- To backup: use Chrome's sync or export feature (coming soon)

## TIPS

- Long press on any item card to share
- Use tags like "important", "documents", "keys" for quick search
- The Mind Map screen gives you a bird's eye view of everything
- Set a reminder before a trip so you don't forget your passport!
